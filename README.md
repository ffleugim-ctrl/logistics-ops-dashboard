# logistics-ops-dashboard
Projeto desenvolvido para análise de KPI's de SLA e produtividade no Centro de Distribuição de Cajamar
 🚀 Otimização de SLA & People Analytics: Operação Cajamar
Este projeto integra indicadores de performance logística (SLA) com métricas de gestão de pessoas para identificar gargalos operacionais e melhorar a eficiência no CD do Mercado Livre em Cajamar.

📌 Contexto do Projeto
Na logística de alta performance, o SLA (Service Level Agreement) é a métrica de ouro. Este dashboard foi desenvolvido para monitorar o cumprimento de prazos e entender como fatores humanos e técnicos influenciam a operação diária.

🛠️ Tecnologias Utilizadas
Python: Limpeza e tratamento da base de dados bruta.

SQL: Estruturação de consultas para análise de performance.

Power BI: Criação de visualizações dinâmicas e indicadores (DAX).

Metodologia Lean: Foco na redução de desperdícios de movimento e tempo.

📊 Principais Indicadores (KPIs)
SLA Geral (%): Média de pacotes entregues rigorosamente no prazo.

Causa Raiz de Atrasos: Identificação visual de gargalos como "Sortter travado" ou "Falta de FTE".

Volume por Turno: Comparativo de produtividade entre as equipes das escalas 6x2.

Lead Time: Tempo médio de processamento por setor (Recebimento, DV, Expedição).

📈 Impacto e Insights
Ao correlacionar o volume de pacotes com os motivos de atraso, o sistema permite:

Apoio à Decisão: Identificar setores que precisam de reforço de equipe em tempo real.

Eficiência Logística: Redução de erros de bipagem e otimização de rotas internas, inspirado em casos reais de melhoria de fluxo observados na operação.
